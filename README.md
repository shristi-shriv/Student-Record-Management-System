# Student-Record-Management-System

Console-based Student Record Management System in Java.

## 🚀 Features
* Add new student records (ID, Name, Course, etc.)
* View a complete list of all students
* Search for a specific student by their unique ID
* Delete or update existing student data

## 🛠️ Prerequisites
* Java Development Kit (JDK) 8 or higher installed on your system.

## 💻 How to Run
Run the `Main.java` file in any Java IDE (like IntelliJ IDEA, Eclipse, or VS Code).
